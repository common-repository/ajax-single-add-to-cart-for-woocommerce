Description
The plugin allows to "Add To Cart" through AJAX without refresh in the single product page.

Features

* The plugin works will all product types [ simple - variable - grouped ] in single page.
* Display the messages after add to cart at the top of the product page.
* You can choose to enable / disable the AJAX feature by product type.

* The plugin works with [Quick View and Buy Now For WooCommerce](https://wordpress.org/plugins/quick-view-and-buy-now-for-woocommerce/) for grouped products which have variable products linked into it.

[Demo](https://demo.grandplugins.com/ajax-single-add-to-cart-for-woocommerce)


[Pro Version](https://grandplugins.com/product/ajax-single-add-to-cart-for-woocommerce/)


* Enable | Disable the AJAX Add To Cart per product.
* Choose to show | hide the "View Cart" link next to "Add to Cart" button after add to cart action.
* Choose to redirect to a custom URL after Add to cart action.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Main Settings at `WooCommerce/Settings/Products/General` Tab
